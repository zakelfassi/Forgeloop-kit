# Signalboard demo fixture

This fixture seeds a fictitious product repo for Forgeloop marketing screenshots and release-proof demos.

It is intentionally static and reviewable:
- all state is canonical repo-local state
- the real loopback service renders it
- screenshots are regenerated by `bin/capture-product-screenshots.sh`

Demo product: **Signalboard** — a collaborative incident timeline and release room for engineering teams.
